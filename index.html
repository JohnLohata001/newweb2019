html

Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.


css


Stands for "Cascading Style Sheet." Cascading style sheets are used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML.


javascript


an object-oriented computer programming language commonly used to create interactive effects within web browsers.


php


(PHP: Hypertext Preprocessor) A scripting language that is widely used to create dynamic Web pages. Combining syntax from the C, Java and Perl languages, PHP code is embedded within HTML pages for server side execution. It is commonly used to extract data out of a database on the Web server and present it on the Web page.
